# Brume

В шапке главной страница сайта содержится логотип студии, главная навигация и телефон для связи со студией. Основное содержимое сайта состоит из раздела преимуществ, раздела «Наши услуги», в котором присутствует дополнительная навигация внутри раздела, также имеется раздел, описывающий этапы работы студии, есть раздел с формой заказа и раздел с отзывами клиентов. В подвале сайта расположен логотип студии, копирайт и ссылки на страницы в социальных сетях.
